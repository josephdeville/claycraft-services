# Manus Landing Page Development Instructions

## Objective
Create a landing page inspired by Coalfire's website design and messaging approach.

## Source Material
- `coalfire-content.md` - Scraped markdown content from coalfire.com
- `coalfire-scrape.json` - Full API response with metadata

## Key Design Elements to Reference

### Brand Positioning
- Tagline: "Secured on all sides"
- Focus: Cybersecurity and compliance services
- Tone: Professional, authoritative, trustworthy

### Page Structure
1. **Hero Section** - Rotating slider with bold headlines and CTAs
2. **Services Grid** - Three-column layout (Advisory, Assessment, Cybersecurity)
3. **Client Stories** - Case study highlights with quotes
4. **Feature Sections** - Alternating image/text blocks
5. **Contact Form** - Lead capture with company/role fields

### Messaging Themes
- AI security and trust engineering
- Regulatory compliance navigation
- Comprehensive assessments (85+ frameworks)
- Elite cybersecurity team (DivisionHex branding)

### Key CTAs
- "Learn more" links to service pages

- Service-specific deep links

## Implementation Notes
- Mobile-responsive design required
- Accessible navigation with skip links
- SEO metadata structure included in scrape
